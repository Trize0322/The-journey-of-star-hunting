    <ul>
      <li>
        <strong>kit</strong> 是特定动物的幼兽，像是河狸、雪貂或鼬鼠。 <sup></sup>
        <ul>
          <li>亦有可能是“kitten”的缩写。</li>
        </ul>
      </li>
      <li>
        推荐的译法为<strong>崽</strong>。
        <ul>
          <li>不建议采用“小”这个翻译，因为容易和“small”、“little”混淆，而且前后缀位置错误。</li>
        </ul>
      </li>
    </ul>

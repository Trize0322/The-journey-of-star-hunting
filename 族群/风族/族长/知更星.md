# 生平
**知更星**是一名<a href="/The-journey-of-star-hunting/世界观/其他/地理位置/湖区领地.html">湖区领地</a>的<a href="/The-journey-of-star-hunting/世界观/族群介绍/风族.html">风族</a><a href="/The-journey-of-star-hunting/世界观/族群分工/族长.html">族长</a>。她曾在<a href="/The-journey-of-star-hunting/族群/星族/星族成员/枯星.html">枯星</a>的领导下担任<a href="/The-journey-of-star-hunting/世界观/族群分工/副族长.html">副族长</a>和<a href="/The-journey-of-star-hunting/世界观/族群分工/武士.html">武士</a>。
**小知更**是<a href="/The-journey-of-star-hunting/族群/风族/武士/响火花.html">响火花</a>和<a href="/The-journey-of-star-hunting/族群/风族/武士/苜蓿泉.html">苜蓿泉</a>的女儿。后来，她成为了<a href="/The-journey-of-star-hunting/世界观/族群分工/学徒.html">学徒</a>**知更爪**，导师是<a href="/The-journey-of-star-hunting/族群/星族/星族成员/枯星.html">枯星</a>。在成为武士**知更熙**后，她爱上了来自<a href="/The-journey-of-star-hunting/世界观/族群介绍/雷族.html">雷族</a>的武士<a href="/The-journey-of-star-hunting/族群/雷族/族长/接骨木星.html">接骨木枝</a>。尽管这是一段不被武士守则容许的恋情，知更熙依旧义无反顾。
但好景不长，在接骨木枝成为雷族的副族长后，她向知更熙提出了分手。后来，知更熙也成为了风族的副族长。她勤勤恳恳地辅佐着自己的族长枯星，这位风族副族长已然忘记了曾经的爱情，现在的她只为了风族而活。
在那场几乎毁灭所有族群的<a href="/The-journey-of-star-hunting/事件/火灾.html">大火</a>之后，<a href="/The-journey-of-star-hunting/世界观/族群介绍/影族.html">影族</a>的族长<a href="/The-journey-of-star-hunting/族群/虚无/魂灵消散/天鹅星.html">天鹅星</a>向所有族群请求帮助。虽然枯星明令禁止风族猫干涉影族的事务，但热心肠的知更熙认为不能置之不理。她和枯星爆发了争吵，随后秘密派遣部分风族猫帮助影族。在影族的<a href="/The-journey-of-star-hunting/世界观/族群分工/巫医.html">巫医学徒</a><a href="/The-journey-of-star-hunting/族群/虚无/魂灵消散/惊爪.html">惊爪</a>宣布他得到了星族的预言之后，枯星终于决定派遣风族猫支援影族。
